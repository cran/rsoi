## rsoi

An R package to download the most up to date - Southern Oscillation
Index - Oceanic Nino Index - North Pacific Gyre Oscillation data - North
Atlantic Oscillation - Arctic Oscillation

## Installation

``` r
install.packages("rsoi")

library(rsoi)
library(ggplot2)
library(tibble)
```
