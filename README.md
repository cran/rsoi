rsoi
------------------
An R package to download the most up to date Southern Oscillation Index and Oceanic Nino Index data. 

Installation
------------------
To install and load the CRAN version

    install.package("rsoi")
    library(rsoi)

Example
------------------

    enso <- download_enso()
    enso


Inspired by
